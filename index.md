# Welcome to DocHub

DocHub is an enterprise-ready, Confluence-style documentation platform built entirely on **Markdown** and **Jekyll**, hosted for free on **GitHub Pages**.

> Write documentation as code. Review it in pull requests. Ship it automatically.

---

## Why DocHub?

| Feature | DocHub | Confluence | SharePoint |
